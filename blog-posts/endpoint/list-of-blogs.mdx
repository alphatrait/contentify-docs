---
title: "List of Blogs"
description: "Get a list of blogs from an external website."
icon: 'list'
---

### Body

Use this if you want to give backlinks to an external website.

<ParamField body="urls" type="string">
    Use the url you want to get the list of blogs from.
</ParamField>


### Response

<ResponseField name="list_of_articles" type="string">
This is the list of blogs from the external website or websites.
</ResponseField>


<RequestExample>

```bash cURL
curl -X POST \
  https://your-domain.com/list-of-articles \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -d '{
    "urls": "https://example-article-url.com"
}'

```

```bash PHP
$url = 'https://your-domain.com/list-of-articles';
$data = array(
    'urls' => 'https://example-article-url.com'
);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, "POST");
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, array(
    'Content-Type: application/json',
    'X-API-Key: YOUR_API_KEY'
));

$response = curl_exec($ch);
curl_close($ch);

```

```bash Python
import requests

url = 'https://your-domain.com/list-of-articles'
headers = {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
}
data = {
    'urls': 'https://example-article-url.com'
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

```

```bash JavaScript
const url = 'https://your-domain.com/list-of-articles';
const headers = {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
};
const data = {
    urls: 'https://example-article-url.com'
};

fetch(url, {
    method: 'POST',
    headers: headers,
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

```
