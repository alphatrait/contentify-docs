

### Body

Get a list of research paper from Arxiv.

<ParamField body="topic" type="string">
  What topic you want to search for. Example: "machine learning"
</ParamField>

### Response

<ResponseField name="hashtag" type="string">
  The list of the hashtags generated for the post.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST \
  <BASE_URL>/arxiv \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -d '{
    "topic": "quantum physics"
}'

```

```bash PHP
<?php
$url = '<BASE_URL>/arxiv';
$data = array(
    'topic' => 'quantum physics'
);

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_CUSTOMREQUEST, "POST");
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, array(
    'Content-Type: application/json',
    'X-API-Key: YOUR_API_KEY'
));

$response = curl_exec($ch);
curl_close($ch);

echo $response;
?>

```

```bash Python
import requests

url = '<BASE_URL>/arxiv'
headers = {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
}
data = {
    'topic': 'quantum physics'
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```bash JavaScript
const url = '<BASE_URL>/arxiv';
const headers = {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
};
const data = {
    topic: 'quantum physics'
};

fetch(url, {
    method: 'POST',
    headers: headers,
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));


```
