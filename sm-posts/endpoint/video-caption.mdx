

### Body

This endpoint will generate social media captions from a description.

<ParamField body="video_description" type="string">
    This is the video description you want to generate social media captions for.
</ParamField>


<ParamField body="tone" type="string">
    How should the social media caption sound like? (e.g. funny, serious, etc.)
</ParamField>

<ParamField body="brand_description" type="string">
    This is the description of your brand.
</ParamField>

<ParamField body="guildelines" type="string">
    Any guidelines you want our model to follow? (e.g. no profanity, etc.)
</ParamField>


### Response

<ResponseField name="caption" type="string">
    This is the generated social media caption.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST \
  <BASE_URL>/video-caption \
  -H 'Content-Type: application/json' \
  -H 'X-API-Key: YOUR_API_KEY' \
  -d '{
    "video_description": "Your video description",
    "tone": "Friendly",
    "guidelines": "Some guidelines",
    "brand_description": "Brand X"
}'


```
